# SG-Acad
## Setup:
1. Clone this repository
2. Run the following commands in terminal:
```
cd sgacad/server
npm i 
nodemon start
```
3. Open another concurrent terminal and run:
```
cd sgacad/client
npm i 
npm run dev
```
4. Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
